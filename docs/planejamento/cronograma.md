# Planejamento

## Introdução

Este documento apresenta o cronograma da disciplina de Interação Humano-Computador para o semestre de 2026.1. Nele, estão contidos todos os planejamentos com as atividades a serem realizadas pela equipe ao longo do projeto, indicando os respectivos responsáveis tanto pelo desenvolvimento quanto pela revisão de cada artefato gerado.
