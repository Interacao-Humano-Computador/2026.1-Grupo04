    Ferramentas

    Introdução

    Para viabilizar a execução do projeto de análise do portal Hemocentro sob a ótica de IHC, a equipe selecionou um conjunto de tecnologias que facilitam a colaboração síncrona e assíncrona. A escolha dessas ferramentas visa otimizar o fluxo de trabalho, desde a comunicação interna até a prototipagem e documentação final, garantindo que os artefatos produzidos mantenham um padrão de qualidade e acessibilidade. A Tabela 1 detalha o ecossistema de software utilizado, especificando o papel de cada solução no ciclo de vida do projeto.

    Tabela 1 - Ferramentas Utilizadas no Projeto

    Logo Ferramenta Finalidade
    GitHub Hospedagem do código-fonte, controle de versão e publicação da documentação via GitHub Pages.1
    Telegram Canal secundário para comunicação rápida e troca de arquivos com a equipe e o docente.2
    Google Meet Realização de reuniões remotas, gravações de apresentações e chamadas de vídeo.3
    Miro Colaboração visual para criação de mapas mentais, fluxogramas e brainstorms iniciais.4
    Figma Desenho de interfaces, wireframes e prototipagem de alta fidelidade para IHC.5
    MkDocs Framework para geração de sites de documentação técnica estática.6
    Visual Studio Code Ambiente de desenvolvimento principal para edição dos arquivos Markdown e YAML.7
    WhatsApp Principal meio de comunicação instantânea e coordenação de tarefas diárias.8
    YouTube Armazenamento e compartilhamento público dos vídeos e apresentações do grupo.9
    Google Planilhas Gestão do cronograma de atividades, controle de prazos e horários.10
    Google Docs Redação colaborativa e revisão de textos antes da transposição para o MkDocs.11
    ChatGPT Suporte na pesquisa de conceitos, auxílio em scripts e revisão gramatical.12
